# ChatGPT Reviews Analysis

This project analyzes ChatGPT user reviews using Python.

## Key Features
- Exploratory Data Analysis (EDA)
- Rating distribution visualization
- Sentiment Analysis using TextBlob

## Tools Used
- Python
- Pandas
- Matplotlib
- TextBlob
- Google Colab / VS Code

## Outcome
Classified user reviews into Positive, Negative, and Neutral sentiments.
